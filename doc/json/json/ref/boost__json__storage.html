Description

This interface is modeled similarly to std::pmr::memory_resource with some notable differences:

  • Instances may be reference counted.
  • The function is_equal is implemented as a non-virtual member which does not require RTTI or typeinfo.
  • The function need_free is provided to allow the implementation to optionally inform callers that calls to deallocate memory are not required.
