storage_ptr::~storage_ptr

Destructor.

Synopsis 
~storage_ptr();
Description

This releases the pointed-to storage. If the storage is reference counted and this is the last reference. the storage object is destroyed. If the storage does not require deallocation, all memory allocated using this storage is invalidated.

Complexity

Constant.

Exception Safety

No-throw guarantee.
