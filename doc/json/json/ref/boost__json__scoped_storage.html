scoped_storage

Constructor.

Description

This wrapper enables the caller to construct a storage objects whose lifetime is controlled by the lifetime of the wrapper instead of a reference counted.

Example

This example creates a pool with bounded lifetime and uses it to parse a JSON, then print it to std::cout. 

{
    scoped_storage<pool> sp;

    auto jv = parse( str, sp );
}
