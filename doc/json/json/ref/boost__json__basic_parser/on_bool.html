basic_parser::on_bool

Called when a bool is parsed.

Synopsis 
void
on_bool(
    bool b,
    error_code& ec);
Description

This function is called when a boolean value is encountered.

Parameters

Name

Description

b

The boolean value.

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
