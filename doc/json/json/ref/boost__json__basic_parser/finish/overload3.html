basic_parser::finish (3 of 4 overloads)

Parse JSON incrementally.

Synopsis 
void
finish(
    error_code& ec);
Description

This function finishes parsing the current JSON, emitting SAX parsing events by calling the derived class. The parse is finalized according to the current state, which includes all previously parsed data since the last reset.

The caller uses this function to inform the parser that there is no more serialized JSON available. If a complete JSON is not available the error is set to indicate failure.

Complexity

Constant.

Parameters

Name

Description

ec

Set to the error, if any occurred.
