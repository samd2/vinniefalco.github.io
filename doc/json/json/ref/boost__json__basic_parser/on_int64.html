basic_parser::on_int64

Called when a std::int64_t is parsed.

Synopsis 
void
on_int64(
    int64_t i,
    error_code& ec);
Description

This function is called when a suitable number is parsed.

Parameters

Name

Description

i

The number encountered.

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
