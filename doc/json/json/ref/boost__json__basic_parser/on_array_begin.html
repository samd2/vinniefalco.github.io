basic_parser::on_array_begin

Called when the beginning of an array is encountered.

Synopsis 
void
on_array_begin(
    error_code& ec);
Description

This function is called during parsing when a new array is started.

Parameters

Name

Description

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
