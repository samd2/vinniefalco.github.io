basic_parser::on_object_end

Called when the end of the current object is encountered.

Synopsis 
void
on_object_end(
    error_code& ec);
Description

This function is called during parsing after all of the elements of an object have been parsed, and the closing brace for the object is reached.

Parameters

Name

Description

ec

The error, if any, which will be returned by the current invocation of write_some, write, or finish.
