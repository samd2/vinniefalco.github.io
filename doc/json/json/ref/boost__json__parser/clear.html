parser::clear

Discard all parsed JSON results.

Synopsis 
void
clear();
Description

This function destroys all intermediate parsing results, while preserving dynamically allocated internal memory which may be reused on a subsequent parse.

Remarks

After this call, it is necessary to call start to parse a new JSON incrementally.
