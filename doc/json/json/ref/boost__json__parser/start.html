parser::start

Start parsing JSON incrementally.

Synopsis 
void
start(
    storage_ptr sp = {});
Description

This function must be called once manually before parsing a new JSON incrementally; that is, when using write_some, write, or finish.

Parameters

Name

Description

sp

A pointer to the storage to use. The parser will acquire shared ownership of the storage object.
