value::if_bool (2 of 2 overloads)

Return a pointer to the underlying bool, or nullptr.

Synopsis 
bool const*
if_bool() const;
Description

If is_bool() is true, returns a pointer to the underlying bool, otherwise returns nullptr.

Complexity

Constant.

Exception Safety

No-throw guarantee.
