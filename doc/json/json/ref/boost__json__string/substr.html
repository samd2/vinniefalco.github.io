string::substr

Return a substring.

Synopsis 
string
substr(
    std::size_t pos = 0,
    std::size_t count = npos) const;
Description

Returns a substring.

Exception Safety

Strong guarantee.

Return Value

A string containing {data() + pos, std::min(count, size() - pos)).

Parameters

Name

Description

pos

The index to being the substring at. The default argument for this parameter is 0.

count

The length of the substring. The default argument for this parameter is npos.

Exceptions

Type

Thrown On

std::out_of_range

pos > size()
