string::string (13 of 15 overloads)

Constructor.

Synopsis 
string(
    std::initializer_list< char > init,
    storage_ptr sp = {});
Description

Construct the contents with those of the initializer list init.

Complexity

Linear in init.size().

Exception Safety

Strong guarantee.

Remarks

Calls to storage::allocate may throw.

Parameters

Name

Description

init

The initializer list to copy from.

sp

An optional pointer to the storage to use. The container will acquire shared ownership of the storage object. The default argument for this parameter is {}.

Exceptions

Type

Thrown On

std::length_error

init.size() > max_size().
