string::string (5 of 15 overloads)

Constructor.

Synopsis 
string(
    string const& other,
    std::size_t pos,
    std::size_t count = npos,
    storage_ptr sp = {});
Description

Construct the contents with a copy of the substring {pos, pos+count) of other. If the requested substring lasts past the end of the string, or if count == npos, the resulting substring is {pos, other.size()).

Complexity

Linear in count.

Exception Safety

Strong guarantee.

Remarks

Calls to storage::allocate may throw.

Parameters

Name

Description

other

The string to use as a source to copy from.

pos

The starting character position to copy from.

count

The number of characters to copy.

sp

An optional pointer to the storage to use. The container will acquire shared ownership of the storage object. The default argument for this parameter is {}.

Exceptions

Type

Thrown On

std::out_of_range

pos >= other.size().
