string::assign (10 of 10 overloads)

Assign characters to a string.

Synopsis 
string&
assign(
    string_view s,
    std::size_t pos,
    std::size_t count = npos);
Description

Replaces the contents with the characters from the substring {pos, pos+count) of s. If count == npos, if count is not specified, or if the requested substring lasts past the end of the string, the resulting substring is {pos, s.size()). The substring can contain null characters.

Complexity

Linear in count.

Exception Safety

Strong guarantee. Calls to storage::allocate may throw.

Parameters

Name

Description

s

The string view to copy from.

pos

The starting character position to copy from.

count

The number of characters to copy.

Exceptions

Type

Thrown On

std::out_of_range

pos >= s.size()

std::length_error

count > max_size().
