is_storage

Metafunction to determine if a type meets the requirements of Storage.

Synopsis

Defined in header <boost/json/storage.hpp> 

template<
    class T>
using is_storage = see-below;
Description

This type alias is std::true_type if the type T satisfies the syntactic requirements of Storage, otherwise it is equivalent to std::false_type.

Exemplar

For the following declaration, is_storage<Storage> is std::true_type: 

struct Storage
{
    static constexpr std::uint64_t id = 0;
    static_constexpr bool need_free = true;

    void* allocate( std::size_t bytes, std::size_t align );
    void deallocate( void* p, std::size_t bytes, std::size_t align );
};
