value_cast

Convert a JSON value to another type T.

Synopsis

Defined in header <boost/json/value_cast.hpp> 

template<
    class T>
auto
value_cast(
    value const& jv);
Description

This function attempts to convert the value to the given type T. It handles these cases, in order of decreasing priority:

  • If value_cast_traits contains a specialization for T, then the function returns T::from_json(jv).
  • If T is numeric, the function returns number_cast<T>(jv).
  • If T is any of the other types that are directly represented in a value (object, array, string, or bool), and the kind of value is suitable then a T const& is returned. If the kind is not suitable then an exception is thrown.
  • Otherwise, if T matches a generic container known to the implementation it is converted.
Constraints 
has_value_cast< T >::value
Exception Safety

Strong guarantee.
