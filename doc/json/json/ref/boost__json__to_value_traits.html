to_value_traits

Customization point to construct a JSON value from a user-defined type.

Synopsis

Defined in header <boost/json/traits.hpp> 

template<
    class T>
struct to_value_traits
Description

This class template is used by the implementation as needed to construct a value from a user-defined type T. The customization point is used by declaring a specialization for the type T and providing a public static member function with this signature: 

void assign( value&, T const& t )

The specialization must be in the same namespace as the library.

Example

This example declares a struct T and specializes the trait to provide a means of construction: 

struct T
{
    int i;
    bool b;
};

// Specialization of to_value_traits must be in the
// same namespace of the library in order to compile.

namespace boost {
namespace json {
template<>
struct to_value_traits< T >
{
    static void assign( value& jv, T const& t )
    {
        jv = { t.i, t.b };
    }
};
} // namespace json
} // naemspace boost
Remarks

Member function to_json or specializations of the to_value_traits template should never be invoked directly. Callers should instead use the to_value function, even when implementing customization points.

Template Parameters

Type

Description

T

The type for which the conversion should be customized.
