object::contains

Check if the container contains an element with a specific key.

Synopsis 
bool
contains(
    key_type key) const;
Description

This function returns true if an element with the specified key exists, otherwise returns false.

Complexity

Constant on average, worst case linear in size().

Exception Safety

No-throw guarantee.

Parameters

Name

Description

key

The key of the element to find.

See Also

find.
