is_special

Return true if the scheme string is a special scheme.

Synopsis

Defined in header <boost/url/scheme.hpp> 

bool
is_special(
    string_view s);
Description

The list of special schemes is as follows: ftp, file, http, https, ws, wss.

Parameters

Name

Description

s

The string constant to check

Return Value

true if the scheme is special
