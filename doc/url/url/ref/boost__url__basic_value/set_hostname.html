basic_value::set_hostname

Set the hostname.

Synopsis 
basic_value&
set_hostname(
    string_view s);
Description

The hostname is set to the specified string, replacing any previous hostname:

  • If the string is empty, the hostname is cleared. If the hostname was the last remaining portion of the authority, then the entire authority is removed including the leading double slash ("//"), else
  • If the string is a valid IPv6Address, the hostname is set to the literal string and host will return host_type::ipv6, else
  • If the string is a valid IPv6Future, the hostname is set to the literal string and host will return host_type::ipvfuture, else
  • If the string is a valid IPv4Address, the hostname is set to the literal string and host will return host_type::ipv4, otherwise
  • The hostname is set to the specified string. Any special or reserved characters in the string are automatically percent-encoded.

If the URL previously did not contain an authority (has_authority returns false), and s is not empty, then the authority is added including a leading double slash ("//").

ABNF 
IPv4address   = dec-octet "." dec-octet "." dec-octet "." dec-octet

IPv6address   =                            6( h16 ":" ) ls32
              /                       "::" 5( h16 ":" ) ls32
              / [               h16 ] "::" 4( h16 ":" ) ls32
              / [ *1( h16 ":" ) h16 ] "::" 3( h16 ":" ) ls32
              / [ *2( h16 ":" ) h16 ] "::" 2( h16 ":" ) ls32
              / [ *3( h16 ":" ) h16 ] "::"    h16 ":"   ls32
              / [ *4( h16 ":" ) h16 ] "::"              ls32
              / [ *5( h16 ":" ) h16 ] "::"              h16
              / [ *6( h16 ":" ) h16 ] "::"

IPvFuture     = "v" 1*HEXDIG "." 1*( unreserved / sub-delims / ":" )
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

s

The string to set. This string may contain any characters, including nulls.
