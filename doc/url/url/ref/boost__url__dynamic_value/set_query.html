dynamic_value::set_query

(Inherited from basic_value)

Set the query.

Synopsis 
basic_value&
set_query(
    string_view s);
Description

Sets the query of the URL to the specified plain string:

  • If the string is empty, the query is cleared including the leading question mark ('?'), otherwise:
  • If the string is not empty, the query is set to the given string, with a leading question mark added. Any special or reserved characters in the string are automatically percent-encoded.
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

s

The string to set. This string may contain any characters, including nulls.
