dynamic_value::set_encoded_fragment

(Inherited from basic_value)

Set the fragment.

Synopsis 
basic_value&
set_encoded_fragment(
    string_view s);
Description

Sets the fragment of the URL to the specified encoded string:

  • If the string is empty, the fragment is cleared including the leading hash mark ('#'), otherwise:
  • If the string is not empty, the fragment is set to given string, with a leading hash mark added. The string must meet the syntactic requirements of fragment otherwise an exception is thrown.
ABNF 
fragment      = *( pchar / "/" / "?" )
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

s

The string to set.

Exceptions

Type

Thrown On

std::exception

invalid string.
