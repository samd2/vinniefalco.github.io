dynamic_value::set_scheme

(Inherited from basic_value)

Set the scheme.

Synopsis 
basic_value&
set_scheme(
    string_view s);
Description

This function sets the scheme to the specified string:

  • If the string is empty, any existing scheme is removed along with the trailing colon (':'), otherwise:
  • The scheme is set to the string, which must contain a valid scheme. A trailing colon is automatically added.
ABNF 
scheme        = ALPHA *( ALPHA / DIGIT / "+" / "-" / "." )
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

s

The scheme to set. This string must not include a trailing colon, otherwise an exception is thrown.

Exceptions

Type

Thrown On

std::exception

invalid scheme.
