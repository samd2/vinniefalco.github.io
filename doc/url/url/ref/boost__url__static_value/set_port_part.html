static_value::set_port_part

(Inherited from basic_value)

Set the port.

Synopsis 
basic_value&
set_port_part(
    string_view s);
Description

The port of the URL is set to the specified string.

  • If the string is empty, the port is cleared including the leading colon (':'). If the port was the last remaining portion of the authority, then the entire authority is removed including the leading double slash ("//"). Otherwise,
  • If the string is not empty then the port is set to the given string, which must have a starting colon. If the URL previously did not contain an authority (has_authority returns false), then the authority is added including the leading double slash ("//"). The string must meet the syntactic requirements of port-part otherwise an exception is thrown.
ABNF 
port-part     = [ ':' *DIGIT ]
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

s

The string to set.
