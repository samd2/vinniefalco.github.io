static_value::set_encoded_query

(Inherited from basic_value)

Set the query.

Synopsis 
basic_value&
set_encoded_query(
    string_view s);
Description

Sets the query of the URL to the specified encoded string:

  • If the string is empty, the query is cleared including the leading question mark ('?'), otherwise:
  • If the string is not empty, the fragment is set to given string, with a leading question mark added. The string must meet the syntactic requirements of query otherwise an exception is thrown.
ABNF 
query         = *( pchar / "/" / "?" )
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

s

The string to set.

Exceptions

Type

Thrown On

std::exception

invalid string.
