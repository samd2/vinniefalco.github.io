static_value::capacity

(Inherited from basic_value)

Return the number of characters that may be stored without a reallocation.

Synopsis 
std::size_t
capacity() const;
Description

This function returns the maximum number of characters which may be stored in the URL before a reallocation is necessary.

Exception Safety

No-throw guarantee.
