static_value::scheme

(Inherited from basic_value)

Return the scheme.

Synopsis 
string_view
scheme() const;
Description

If there is no scheme, an empty string is returned. Otherwise the scheme is returned, without a trailing colon (':').

Exception Safety

No-throw guarantee.
