basic_value::params_type::at

Return the param matching the given key.

Synopsis 
template<
    class Allocator = std::allocator<char>>
string_type< Allocator >
at(
    string_view key,
    Allocator const& a = {}) const;
Description
Exception Safety

Strong guarantee. Calls to allocate may throw.

Parameters

Name

Description

a

An optional allocator the returned string will use. If this parameter is omitted, the default allocator is used, and the return type of the function becomes std::string.

Return Value

A std::basic_string using the specified allocator.
