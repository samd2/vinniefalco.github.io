http::basic_fields::set (1 of 2 overloads)

Set a field value, removing any other instances of that field.

Synopsis 
void
set(
    field name,
    string_param const& value);
Description

First removes any values with matching field names, then inserts the new field value.

Parameters

Name

Description

name

The field name.

value

The value of the field, as a string_param

Return Value

The field value.
