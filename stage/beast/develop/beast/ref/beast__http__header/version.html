http::header::version

The HTTP-version.

Synopsis 
unsigned version = 11;
Description

This holds both the major and minor version numbers, using these formulas: 

unsigned major = version / 10;
unsigned minor = version % 10;

Newly constructed headers will use HTTP/1.1 by default.
