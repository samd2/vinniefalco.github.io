websocket::teardown_tag

Tag type used to find websocket::teardown and websocket::async_teardown overloads.

Synopsis

Defined in header <beast/websocket/teardown.hpp> 

struct teardown_tag
Description

Overloads of websocket::teardown and websocket::async_teardown for user defined types must take a value of type websocket::teardown_tag in the first argument in order to be found by the implementation.

Convenience header <beast/websocket.hpp>
