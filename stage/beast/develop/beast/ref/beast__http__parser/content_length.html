http::parser::content_length

(Inherited from http::basic_parser)

Returns the optional value of Content-Length if known.

Synopsis 
boost::optional< std::uint64_t >
content_length() const;
Description
Remarks

The return value is undefined unless http::basic_parser::is_header_done would return true.
