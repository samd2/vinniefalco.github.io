basic_stream_socket::write_limit

Set the number of bytes allowed to be written per second.

Synopsis 
void
write_limit(
    std::size_t bytes_per_second);
Description

The limit will take effect in the next measured time interval (currently set to 3 seconds).

Parameters

Name

Description

bytes_per_second

The maximum number of bytes the implementation should attempt to write per second. A value of zero indicates no limit.
