basic_stream_socket::basic_stream_socket (4 of 10 overloads)

Construct and open a basic_stream_socket.

Synopsis 
basic_stream_socket(
    executor_type const& ex,
    protocol_type const& protocol);
Description

This constructor creates and opens a stream socket. The socket needs to be connected or accepted before data can be sent or received on it.

Parameters

Name

Description

ex

The executor which the stream socket will use to dispatch handlers for any asynchronous operations performed on the socket. Currently, only executors that return net::io_context& from ex.context() are supported.

protocol

An object specifying protocol parameters to be used.

Remarks

This function does not participate in overload resolution unless: * std::is_convertible<ExecutionContext&, net::execution_context&>::value is true, and * std::is_constructible<executor_type, typename ExecutionContext::executor_type>::value is true.
