basic_timed_stream::write_expires_after

Set the timeout for the next logical write operation.

Synopsis 
void
write_expires_after(
    std::chrono::nanoseconds expiry_time);
Description

This sets the write timer to expire after the specified amount of time has elapsed. If the write timer expires and an asynchronous write operation is outstanding, the stream will be closed and the operation will complete with the error timeout. If no write operation is outstanding, then the stream will be closed when the next asynchronous write operation is initiated. In this case the operation will complete immediately with the error timeout.

The timer applies collectively to all asynchronous writes initiated after the expiration is set.

Parameters

Name

Description

expiry_time

The amount of time after which a logical write operation should be canceled due to timeout.

Remarks

Changing or disabling the write timer's expiration while a write operation is outstanding results in undefined behavior.
