basic_timed_stream::read_expires_never

Disable the timeout for logical read operations.

Synopsis 
void
read_expires_never();
Description

This disables the read timer used by basic_timed_stream::async_read_some.

Remarks

Changing or disabling the read timer's expiration while a read operation is outstanding results in undefined behavior.
