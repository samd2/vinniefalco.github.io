Description

This class template is parameterized on the executor type to be used for all asynchronous operations. This achieves partial support for [P1322]. The default template parameter uses a strand for the next layer's executor.

Unlike other stream wrappers, the underlying socket is accessed through the stranded_stream::socket member function instead of next_layer. This causes stranded_stream to be returned in calls to get_lowest_layer.
