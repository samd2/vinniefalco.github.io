basic_multi_stream::operator= (2 of 2 overloads)

Converting Assignment.

Synopsis 
template<
    class OtherProtocol,
    class OtherExecutor>
basic_multi_stream&
operator=(
    basic_multi_stream< OtherProtocol, OtherExecutor >&& other);
Description

The behavior of converting assignment while asynchronous operations are pending is undefined.

Parameters

Name

Description

other

The object to convert-assign from. After this constructor returns, the object will be as-if moved-from.

Remarks

This constructor does not participate in overload resolution unless OtherProtocol is implicitly convertible to Protocol, and OtherExecutor is implicitly convertible to Executor.
