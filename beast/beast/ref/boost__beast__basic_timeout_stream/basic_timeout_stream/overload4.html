basic_timeout_stream::basic_timeout_stream (4 of 5 overloads)

Construct the stream with an executor and existing socket.

Synopsis 
basic_timeout_stream(
    executor_type const& ex,
    net::basic_stream_socket< Protocol >&& socket);
Description

This constructor creates a timeout stream by taking ownership of an already existing socket.

Parameters

Name

Description

ex

The executor which stream will use to dispatch handlers for any asynchronous operations performed on the underlying socket. Currently, only executors that return a net::io_context& from ex.context() are supported.

socket

The socket object to construct with, which becomes the next layer of the timeout stream. Ownership of this socket is transferred by move.
