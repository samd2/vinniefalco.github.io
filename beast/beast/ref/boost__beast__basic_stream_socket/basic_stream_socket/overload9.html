basic_stream_socket::basic_stream_socket (9 of 10 overloads)

Move-construct a basic_stream_socket from another.

Synopsis 
basic_stream_socket(
    basic_stream_socket&& other);
Description

This constructor moves a stream socket from one object to another.

The behavior of moving a stream socket while asynchronous operations are outstanding is undefined.

Parameters

Name

Description

other

The other basic_stream_socket object from which the move will occur.

Remarks

Following the move, the moved-from object is in the same state as if constructed using the basic_stream_socket(ExecutionContext&) constructor.
