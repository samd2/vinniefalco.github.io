http::message::method (1 of 2 overloads)

(Inherited from http::header)

Return the request-method verb.

Synopsis 
verb
method() const;
Description

If the request-method is not one of the recognized verbs, http::unknown is returned. Callers may use http::header::method_string to retrieve the exact text.

Remarks

This function is only available when isRequest == true.
