async_completion::async_completion

Constructor.

Synopsis 
async_completion(
    CompletionToken& token);
Description

The constructor creates the concrete completion handler and makes the link between the handler and the asynchronous result.

Parameters

Name

Description

token

The completion token. If this is a regular completion handler, copies may be made as needed. If the handler is movable, it may also be moved.
