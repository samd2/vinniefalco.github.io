Description

This object implements a dynamic buffer with a fixed size output area, not dynamically allocated, and whose input sequence is always length zero. Bytes committed from the output area to the input area are always discarded. This is useful for calling interfaces that require a dynamic buffer for storage, but where the caller does not want to retain the data.
